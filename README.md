Deskripsi Program:

- 🚗 Program 1: Program Sistem Parkir Otomatis Mall "Prienal" 🚗

Program Sistem Parkir Otomatis Mall "Prienal" adalah program berbasis konsol menggunakan C++ yang dirancang untuk membantu petugas parkir mencatat kendaraan masuk, menampilkan daftar kendaraan beserta durasi parkir secara real-time, menghitung biaya otomatis, serta memberi notifikasi jika parkiran penuh. Program ini mengimplementasikan struct, pointer, vector, dan function sebagai komponen utamanya.

- 🔠 Program 2: Game Tebak Kata 🔠

Game Tebak Kata adalah program konsol berbasis C++. Dimana pemain harus menebak kata huruf per huruf sebelum nyawa habis 
dengan pilihan tiga level kesulitan (Easy, Medium, dan Hard). Dilengkapi fitur hint, sistem skor, dan leaderboard top 5
yang diurutkan otomatis. Program terdiri dari dua file: TebakKata.h dan TebakKata.cpp.
    
- 🏪 Program 3: Sistem Manajemen Inventaris Toko 🏪

Program ini merupakan sistem manajemen inventaris berbasis C++ yang dirancang untuk membantu pengelolaan data barang toko secara efisien melalui antarmuka menu di terminal. Program ini memungkinkan pengguna untuk menambah, memantau, dan memperbarui data barang dalam sebuah toko. Untuk memperbarui stok barang, program menggunakan pointer agar dapat langsung mengakses alamat memori yang dituju. Seluruh data barang disimpan dan dikelola secara dinamis menggunakan vector.

- 🎮 Program 4: Game Minesweeper ++ 🎮
  
Game ini adalah game tebak-tebakan di mana pemain harus menemukan semua bom yang tersembunyi di dalam sebuah papan kotak-kotak, sampai menginjak (membuka) salah satunya. 
Cara mainnya:
- Pemain bebas pilih ukuran papan, dari 4×4 sampai 10×10, dan berapa banyak bom yang mau ditanam.
- Setiap giliran, pemain bisa buka kotak untuk lihat isinya atau pasang tanda F di kotak yang dicurigai ada bom.
- Kalau buka kotak yang isinya bom, langsung game over.
- Kalau berhasil menandai semua bom dengan tepat tanpa satu pun salah posisi, pemain menang.

- 🗺️ Program 5: Game Tebak Nama Negara 🗺️

Game Tebak Nama Negara adalah game berbasis teks (console game) di mana pemain harus menebak nama negara dari petunjuk huruf yang sebagian disembunyikan oleh sensor (_) secara acak. Game ini dilengkapi dengan sistem skor otomatis dan fitur lanjut bermain untuk mengumpulkan poin tertinggi.
